## Installation

```bash
cd backend
npm install
cd ../frontend
npm install
```

## Usage

### Backend

Create a `.env` file in the `backend` directory with the following content:

```env
MONGO_URI=mongodb://127.0.0.1:27017/gamigami
JWT_SECRET=your_secret
PORT=5000
```

Run the backend server:

```bash
cd backend
node server.js
```

### Frontend

Run the frontend:

```bash
cd frontend
npm run dev # for development
```
